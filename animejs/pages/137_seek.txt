seek()
Updates the
currentTime
of the animation and advances it to a specific time.
animation.
seek
(time, muteCallbacks);
Parameters
Name
Type
Description
time
Number
The new
currentTime

muteCallbacks=false
(opt)
Boolean
If

The animation itself
